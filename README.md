# Кластерный анализ посетителей фитнес-клуба
---
## О проекте
Кластерный анализ поведения клиентов фитнес-клуба 
с целью персонализации маркетинга и снижения оттока.
---
## Данные
[Gym Members Exercise Dataset](https://www.kaggle.com/datasets/valakhorasani/gym-members-exercise-dataset) — 
973 записи, 15 признаков: возраст, пол, тип тренировки, 
пульс, калории, процент жира и др.
---
## Методология
- Удалены коррелирующие и нефизиологические признаки
- Два эксперимента: с типом тренировки и без
- KMeans, выбор k через метод силуэта и метод локтя
- Валидация через сравнение с Experience_Level
- Визуализация через PCA
---
## Результаты
Выявлено две принципиально разные группы:
- **Продвинутые (кластер 2)** — высокая активность, 
  низкий процент жира, преимущественно мужчины
- **Остальные три кластера** — схожи по физиологии, 
  различаются только типом тренировки
Тип тренировки оказался основным разделяющим признаком в данных.
